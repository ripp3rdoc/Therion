# Therion

## What is Therion?
Therion is a simple bash script automating reconnaissance on WebApps for Penetration Testers and Bug Bounty Hunters.



## Features:
- Subdomain Enumeration.
- Directory Brute Forcing.
- Port Scanning.
- Data Scraping.
- File Creation & Information Categorization.

## Requirements
- nmap
- subjack 
- assetfinder 
- httprobe

## Basic Usage: 
```
~# ./Therion.sh <URL>
```
